# Net-browser positioning (MVP)

## Net-browser nima?
- **Everyday browser** UX: oddiy user uchun (tab/address bar)
- **Privacy-first roadmap**: default policy, tracker/cookie partitioning keyin core’da
- **Transparent control**: keyin Privacy Dashboard (WebUI)

## Chrome vs Net-browser
- Chrome: general-purpose, privacy ko'pincha ecosystem tension ichida
- Net-browser: **privacy by default** + **policy engine ajratilgan** + minimal telemetry

## Brave vs Net-browser
- Brave: Shields bilan kuchli built-in blocking
- Net-browser: **isolation + policy modes** markazda (Work/Bank/Strict/Compat)
  va arxitektura “clean boundaries”ga urg'u

## Tor vs Net-browser
- Tor: anonymity-first (kuchli), lekin kundalik usability qimmat
- Net-browser: **kundalik privacy** (tezroq, ko'proq sayt ishlaydi), Tor-level anonymity va'da qilmaydi (halol threat model)
